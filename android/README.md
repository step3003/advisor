# Advisor — Android-приложение

Полноценный клиент под Android: **кабинет (тот же веб-UI в WebView) + вход в
аккаунт + захват сообщений**. Один аккаунт со всеми устройствами; данные текут
на твой сервер.

## Что умеет
- **Вход в аккаунт** (логин/пароль) — тот же аккаунт, что в вебе.
- **Кабинет**: показывает веб-интерфейс Advisor (отчёты, план/факт, операции,
  SMS-парсер и т.д.) — автоматически залогинен.
- **Захват SMS**: входящие SMS пересылаются на сервер и разбираются по шаблонам.
- **Захват уведомлений**: пуши выбранных приложений (напр. банковских) тоже
  пересылаются и разбираются теми же шаблонами — ловит онлайн-оплаты, о которых
  банк шлёт пуш, а не SMS. Требует «Доступ к уведомлениям».
- Переключатели в «Настройки захвата»: что ловить, фильтры отправителя/приложений.

Разбор — на сервере (раздел «SMS-парсер» в кабинете): и SMS, и уведомления идут
на один эндпоинт `POST /api/ingest/sms` (для уведомления `sender` = package
приложения, `text` = текст пуша).

## Сборка APK
Проще всего — через GitHub Actions (workflow **Build Android APK**): пуш в ветку
или запуск вручную → артефакт `advisor-sms-apk` → `app-debug.apk`. Локально:
Android Studio (открыть папку `android/`) или `./gradlew assembleDebug`.

## Установка и настройка
1. Поставить APK (сайдлоад — «установка из неизвестных источников»).
2. Открыть **Advisor**, ввести адрес сервера (`https://…`), логин и пароль → «Войти».
3. Откроется кабинет. Внизу — «Настройки захвата»:
   - включить **Ловить SMS** (и выдать разрешение на SMS при запросе);
   - включить **Ловить уведомления**, указать приложения (напр. `priorbank`) и нажать
     **«Выдать доступ к уведомлениям»** → в системном списке включить Advisor.
4. Настроить шаблоны разбора в кабинете (раздел «SMS-парсер»).

## Ограничения
- Захват (SMS/уведомления) — **только Android**. На iPhone это невозможно (Apple);
  там используется PWA только для просмотра.
- Google Play ограничивает доступ к SMS/уведомлениям — распространение сайдлоадом.
- Некоторые прошивки глушат фоновые сервисы — добавь приложение в исключения
  энергосбережения, если захват перестанет работать.
