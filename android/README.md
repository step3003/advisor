# Advisor SMS — Android-форвардер

Крохотное Android-приложение: ловит входящие банковские SMS и пересылает **сырой
текст** на сервер Advisor (`POST /api/ingest/sms`). Разбор SMS выполняется **на
сервере** по шаблонам, настроенным в кабинете (раздел «SMS-парсер») — приложение
ничего не парсит само.

## Что делает
- Ловит `SMS_RECEIVED` (нужно разрешение `RECEIVE_SMS`).
- Опционально фильтрует по подстроке отправителя (напр. `Bank`).
- Шлёт `{sender, text}` на `<serverUrl>/api/ingest/sms` с `Authorization: Bearer <токен>`.
- Экран настройки: адрес сервера, токен, фильтр, включатель, кнопка «Тест».

## Сборка
В этом окружении APK не собирался (нет Android SDK). Собрать так:

**Вариант A — Android Studio (проще):**
1. Открыть папку `android/` как проект.
2. Дать студии скачать зависимости, затем Run на устройстве/эмуляторе или Build → Build APK.

**Вариант B — CLI:**
1. Установить Android SDK, задать `ANDROID_HOME`.
2. Сгенерировать gradle wrapper: `cd android && gradle wrapper` (нужен установленный Gradle 8.7+).
3. Собрать: `./gradlew assembleDebug` → `app/build/outputs/apk/debug/app-debug.apk`.

## Установка и использование
1. Поставить APK на Android (сайдлоад — включить «установка из неизвестных источников»).
   Google Play жёстко ограничивает доступ к SMS, поэтому распространение — сайдлоадом.
2. Открыть приложение, ввести **URL сервера** (тот же, что открываешь в браузере) и **токен**,
   включить переключатель, нажать «Тест» — должно прийти «Тест успешен».
3. Готово: входящие SMS от банка теперь автоматически превращаются в операции по
   твоим шаблонам; нераспознанные попадают во «Входящие» кабинета.

## Замечания по безопасности
- Приложение хранит токен в `SharedPreferences` (локально на устройстве).
- Трафик идёт на твой сервер по HTTPS (настрой TLS на VPS — Фаза деплоя).
- Разрешение `RECEIVE_SMS` даёт доступ ко всем SMS — устанавливай только свою сборку.

## Ограничения
- Нужен runtime-доступ к SMS (Android 6+ запрашивает разрешение при первом запуске).
- Некоторые прошивки убивают фоновые receiver'ы при агрессивной экономии батареи —
  добавь приложение в исключения энергосбережения, если SMS перестанут приходить.
