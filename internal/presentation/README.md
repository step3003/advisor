# Presentation (СЛОЙ 4) — UI на Fyne

Слой пользовательского интерфейса на **Fyne**. Зависит только от `application`
(usecase-сервисов) и `domain`; к `infrastructure` не обращается (проверяется
`go list -deps`). Все строки — на русском в пакете `i18n` (NFR-8).

## Структура

```
presentation/
├── app/       — окно приложения и навигация по вкладкам (app.Run)
├── screens/   — экраны: План/Факт, Ввод факта, Отчёты, Категории,
│                Повторяющиеся, Настройки; view-model в viewmodel.go (тестируется)
├── widgets/   — переиспользуемые виджеты: ввод суммы, выбор категории,
│                горизонтальная и столбчатая диаграммы (canvas-примитивы, без внешних либ)
└── i18n/      — все строки UI и локале-зависимое форматирование (деньги, даты, %)
```

## Экраны

- **План/Факт** (`balance.go`) — таблица план/факт/отклонение/%/остаток по месяцу,
  итоги, «осталось до конца месяца», задание плана, копирование из прошлого месяца.
- **Ввод факта** (`transactions.go`) — форма операции (тип/дата/категория/сумма/валюта/
  комментарий), список последних операций с редактированием и удалением.
- **Отчёты** (`reports.go`) — период (пресеты + произвольный), расходы по категориям
  и динамика доход/расход по месяцам в виде диаграмм, разбивка по исходным валютам.
- **Категории** (`categories.go`) — дерево категория→подкатегория, создание/
  переименование/архивация/удаление, показ архивных.
- **Повторяющиеся** (`recurring.go`) — шаблоны: создание/редактирование/пауза/удаление.
- **Настройки** (`settings.go`) — валюта по умолчанию, обновление курсов НБ РБ,
  экспорт/импорт (JSON снапшот, CSV за период), статус синхронизации vault + пересканирование.

## Тестирование

Логика подготовки данных вынесена в чистые функции (`screens/viewmodel.go`,
`widgets/chart.go`) и покрыта unit-тестами. Сами Fyne-виджеты не тестируются.

## Платформенный iCloud

Реальный iCloud-контейнер (entitlements, `iCloud.<bundle-id>`, файловый вотчер на
iOS) подключается на этапе сборки iOS/macOS. Слой `Vault` спроектирован так, что
путь к папке — параметр; presentation и домен об iCloud не знают.
